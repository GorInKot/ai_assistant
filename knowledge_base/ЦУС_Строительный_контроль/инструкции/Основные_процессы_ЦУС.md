# ЦУС: карта основных процессов (актуальная)

## Назначение
Файл для быстрой навигации по ключевым процессам в ЦУС и выбора нужной инструкции.

## Процесс 1. Доступ в ЦУС
- Цель: получить доступ к объектам и модулям.
- Вход: заявка на доступ.
- Выход: активная учетная запись с нужными правами.
- Документ: `Шаблон заявки на доступ в ЦУС.docx`.

## Процесс 2. Настройка объекта
- Цель: подготовить исходные данные по объекту.
- Ключевые действия:
  - заполнить карточку объекта;
  - добавить участников и роли;
  - загрузить исходные файлы.
- Модуль: `Информация`.

## Процесс 3. Стройконтроль
- Цель: провести проверки и зафиксировать результаты контроля качества.
- Ключевые действия:
  - создать проверку;
  - выбрать вид контроля качества;
  - заполнить чек-листы;
  - оформить акт проверки/предписание/недостатки/акт устранения.
- Модуль: `Стройконтроль`.

## Процесс 4. Журналы
- Цель: вести журналы работ и контрольных записей.
- Ключевые действия:
  - создать журнал по шаблону;
  - заполнить реквизиты и разделы;
  - обработать замечания;
  - отправить журнал на согласование/подписание.
- Модуль: `Журналы`.

## Процесс 5. Исполнительная документация (ИД)
- Цель: оформить исполнительные документы по выполненным работам.
- Ключевые действия:
  - создать акт (в ИД, из ГПР или из Журналов);
  - заполнить обязательные поля;
  - прикрепить подтверждающие файлы;
  - провести документ и передать в согласование.
- Модули: `Исполнительная документация`, `ГПР`, `Журналы`.

## Процесс 6. Документооборот в СЭД
- Цель: обеспечить маршрутизацию, согласование и подписание.
- Ключевые действия:
  - создать карточку документа;
  - запустить ДО по типу/регламенту/параметрам;
  - обработать статусы и уведомления;
  - выгрузить листы согласования.
- Модуль: `СЭД`.

## Процесс 7. Отчеты и аналитика
- Цель: подготовить отчетность по объекту и контрольным событиям.
- Ключевые действия:
  - сформировать отчет;
  - настроить структуру/фильтры;
  - выгрузить печатную форму.
- Модули: `Отчеты`, `Стройконтроль` (аналитические вкладки).

## Актуальные изменения версии (release notes)
- В `Стройконтроль` добавлены: `Вид контроля качества`, вкладка `Чек-листы`.
- В `Журналы` добавлен импорт/экспорт через Excel-шаблон.
- В ИД расширено повторное использование данных в полях актов.
- Усилен контроль обязательных полей для части актов при проведении.

## Главная связка процессов
`Информация` -> `Стройконтроль` -> `Журналы` -> `ИД` -> `СЭД` -> `Отчеты`.
