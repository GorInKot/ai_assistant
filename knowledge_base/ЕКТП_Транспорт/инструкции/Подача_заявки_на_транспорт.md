# Подача заявок на транспорт в ЕКТП

## Назначение
Процесс нужен для бронирования корпоративного транспорта по служебным задачам.

## Шаги
1. Открыть ЕКТП и выбрать раздел "Транспорт".
2. Нажать "Новая заявка".
3. Указать дату, время, маршрут и цель поездки.
4. Выбрать количество пассажиров и контактное лицо.
5. При необходимости добавить комментарий для диспетчера.
6. Отправить заявку на согласование.
7. Отслеживать статус в разделе "Мои заявки".

## Статусы
- Черновик
- На согласовании
- Подтверждена
- Отклонена
